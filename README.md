# n8n AI Blog Automation Workflow

This project is an automation workflow built with n8n.

## What it does
- Reads keywords from Google Sheets
- Generates blog titles and article content using AI
- Processes JSON output
- Publishes articles to WordPress via REST API
- Updates workflow status automatically

## Stack
- n8n
- JavaScript
- OpenRouter API
- Google Sheets
- WordPress REST API
- Docker

## Files
- `workflow.json`: exported n8n workflow

## Notes
